# Roadmap

## Phase 1: Control Plane Foundation

- monorepo, docs, ADRs, API spec
- organization, project, team, service, environment core models
- change ingestion and deterministic risk scoring
- rollout planning v1
- policy evaluation and audit foundations
- starter integrations registry
- premium frontend shell
- local development environment and tests

## Phase 2: Runtime and Workflow Expansion

- PostgreSQL-backed repositories
- workflow orchestration and long-running job execution
- deployment verification results
- incident linking and service health rollups
- startup bootstrap wizard
- enterprise adoption modes
- graph views and topology drill-down

## Phase 3: Governance Depth

- richer policy packs
- compliance evidence collection and exports
- approval workflows and separation-of-duties
- deployment freeze windows and release trains
- rollout verification against business metrics
- stronger tenant and enterprise administration controls

## Phase 4: Differentiated Intelligence

- simulation lab
- historical incident correlation
- blast-radius prediction refinement
- cost-aware rollout optimization
- AI-assisted explanations built on deterministic platform outputs

## Near-Term Highest-Value Tasks

- add PostgreSQL repository implementation behind existing interfaces
- attach rollout planning to deployment and verification models
- ingest external metadata from GitHub and Kubernetes adapters
- add authn/authz middleware and service-account flows
- model service dependencies and graph queries more deeply
- add web-to-API live data bindings and richer state handling
